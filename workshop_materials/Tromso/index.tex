\section{Welcome to the course!}\label{welcome-to-the-course}

\begin{itemize}
\tightlist
\item
  Welcome to the \texttt{inlabru} workshop!
\item
  The goal of this workshop is to introduce new users to the
  \texttt{inlabru} software for Bayesian spatial modelling using the
  integrated nested Laplace approximation (INLA).
\item
  The workshop is intended for applied statisticians and quantitative
  ecologists who are interested in efficiently modelling spatial data-
  No prior knowledge of \texttt{R-INLA} is required.
\end{itemize}
